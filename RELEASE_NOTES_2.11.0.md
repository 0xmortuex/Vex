Vex 2.11.0 — the big one. Eight feature drops in a single release: a reading suite, a privacy pack, developer tools, persistent AI memory, on-device AI, an MCP client, and AI vision. Existing installs auto-update.

## 🖍 Reading & knowledge (v2.4.0)
- **Persistent highlights** — highlight any text; it reappears every time you revisit the page. Notes per highlight, plus a Highlights panel across all pages.
- **Recall ("memex")** — local full-text search of everything you've read; find a page by its *content*, not just its title.
- **Reading & Accessibility** — dyslexia fonts, color-vision filters, reading ruler, Bionic Reading, RSVP speed reader, Translate Selection.

## 🛡 Privacy hardening (v2.5.0 + v2.8.0)
- **Fingerprint protection** — per-session canvas/WebGL/audio farbling so sites can't fingerprint you.
- **DNS-over-HTTPS** — Cloudflare / Google / Quad9, opportunistic or strict.
- **Privacy Report** — trackers blocked, and **which trackers follow you across multiple sites**.

## 🧰 Developer & power tools (v2.6.0)
- **API client** with a collapsible JSON tree, **Format JSON** for raw-JSON tabs.
- **Responsive Preview** — the page at phone/tablet/laptop/desktop widths side-by-side.
- **Watch This Page** — get alerted when a page changes (restocks, docs, status pages).
- **Wayback archiving** — save a snapshot / open the archived version of any link.

## 🧠 AI (v2.7.0, v2.9.0, v2.10.0, v2.11.0)
- **AI Memory** — tell Vex facts to remember in every chat (works local + cloud, syncs across devices).
- **On-Device AI (WebGPU)** — run a small model entirely on your machine: private, offline, no server. Download a model, flip it on. Falls back to cloud automatically.
- **MCP client** — connect to Model Context Protocol servers, browse and run their tools, and let the **autonomous agent** call them too.
- **Screenshot → Code** — capture a page and have AI rebuild it as HTML / Tailwind / React, with a live preview.
